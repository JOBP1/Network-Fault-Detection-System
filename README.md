  **Network-Fault-Detection-System**
A system that monitors network devices, detects faults, and sends alerts using a virtualized environment.

  **Project Description**
This project is designed to monitor network devices in a virtual environment, detect faults such as device downtime or abnormal response time, and automatically notify the network administrator.

  **Objectives**
- Monitor network devices in real time
- Detect faults such as unreachable hosts
- Generate automatic alerts
- Display device status on a dashboard

  **Tools**
- Python
- Virtual Machines (VirtualBox / VMware)
- Flask (for dashboard)
